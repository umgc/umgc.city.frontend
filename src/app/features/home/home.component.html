Free HTML Code Generator Tool for City Officials


The purpose of this tool is to help city officials create a quick reference page that can answer many of the city's land use zoning questions they may get on a regular basis. This tool will help city officials organize their city's zoning data in a way that enables it to be presented on thier current city's web application in a well organized and easy to read format. City residents will be able to quickly find data and links pertaining to each zone, including development standards, required permits, and links to permit applications (if available).
